== Building

[source,bash]
----
mvn install -DskipTests  # <1>
cd litterbox-web
mvn package spring-boot:repackage
----
<1> Needed for Maven to be able to find the LitterBox dependency in litterbox-web.

Afterwards, you can start the litterbox-web server using
[source,bash]
----
java -jar litterbox-web/target/litterbox-web-${version}.jar
----

Starting the application directly via your IDE from the main class in litterbox-web should also work.
