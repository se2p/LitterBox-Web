# Readme

When using publicly shared Scratch projects as fixtures, the licence has to be like
```
SPDX-FileCopyrightText: 2024 Scratch User "<USERNAME>"

SPDX-License-Identifier: CC-BY-SA-2.0
```

As per [Scratch Terms of Use](https://scratch.mit.edu/terms_of_use) section 4.3 all user-generated content is licenced under the [Create Commons Attribution-ShareAlike 2.0 licence](https://creativecommons.org/licenses/by-sa/2.0/).

Small demo projects created yourself specifically to use as a test fixture should use
```
SPDX-License-Identifier: EUPL-1.2

SPDX-FileCopyrightText: CURRENT_YEAR LitterBox-Web contributors
```
